S-Messenger

FORM: Webpage, Chrome App

ADDRESS: greatcorn.github.io/scom/s-messenger/

LATEST UPDATE: 19.03.2020

CREATION DATE: 02.03.2020

DESCRIPTION: A fully functional WebSocket chat, based on the SCP Foundation and SCP - Containment Breach (S-COM SOFTWARE 1.5). The chat first had a default WebSocket address, from a WebSocket hosting service. Later, the service started providing full functionality to only paid WebSocket addresses. Generally, the chat still works fine with any WSS WebSockets. The chat can also be installed as a Chrome app.

The idea first came as an S-COM SOFTWARE 1.3 HTML recreation. Hosted on the same GitHub repo, greatcorn.github.io/scom/ has a somewhat functional WebSocket chat attempt, while also recreating the looks and feels of the S-COM SOFTWARE 1.3. Later an attempt was made to recreate the later version, 1.5. greatcorn.github.io/scom/scom15.html has a working desktop system, the S-Messenger built into it and the first attempt to create an SCP-themed programming language, SCCOL, in the form of a console.
